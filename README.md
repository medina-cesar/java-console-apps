# Java Console Apps

A collection of Java console applications built to develop core programming skills — covering user input handling, control flow, methods, loops, and array manipulation.

---

## Projects

### Method calculator (`final-practice/`)
A multi-operation calculator supporting addition, subtraction, multiplication, and division across a variable number of inputs.

**Features:**
- Accepts any number of inputs per operation
- Subtraction and division read the first number separately to prevent incorrect results from starting at zero
- Division by zero is caught with a retry prompt — the operation continues without crashing
- Each operation is handled by a dedicated method, keeping logic clean and separated

**Concepts practiced:** methods, parameters, loops, switch statements, input validation

---

### Vending Machine Simulator (`vending-machine/`)
An interactive console vending machine with a full snack menu, live running total, and multi-item selection.

**Features:**
- Displays a formatted snack menu on launch
- Accepts item codes (e.g. A1, B3) and adds the price to a running total
- Allows multiple selections before checking out
- Handles invalid input with a default case
- Prints a formatted final total on exit

**Concepts practiced:** loops, switch statements, formatted output, user input handling

---

### Calculators (`calculators/`)
A set of calculator implementations exploring different control flow approaches.

- **DogYearCalculator** — converts human years to dog years
- **ForLoopCalc** — calculator logic implemented using for loops
- **SwitchCalculator** — calculator logic implemented using switch statements

**Concepts practiced:** arithmetic operations, loops, switch statements, type conversion

---

### Method Practice (`method-practice/`)
A series of programs focused on writing and calling custom methods.

- Finding the maximum value across user inputs
- Computing the average of a set of numbers
- Progressively more complex method implementations across three files

**Concepts practiced:** method definition, return types, parameters, reusability

---

### Array Practice (`array-practice/`)
Programs focused on array traversal and data analysis.

- Finding the largest value in an array
- Counting positive, negative, and zero values
- Iterating through arrays with loops

**Concepts practiced:** arrays, loops, conditionals, data analysis
