🧩 Phase 1: Setup & Fundamentals (Modules 1–2)
🎯 Learning Targets

Install your environment (IntelliJ or VS Code with Java).
Understand the structure of a Java program:
Master variables, data types, and operators.

💻 Practice Tasks

Write a “Hello, Name” program that takes user input and prints output.
Experiment with arithmetic and string concatenation.
Create a calculator program (addition, subtraction, etc.).

🧩 Phase 2: Control Structures (Modules 3–4)
🎯 Learning Targets

if/else, switch
for, while, do-while loops
Nesting and logic operators

💻 Practice Tasks

Write a grade calculator or temperature converter using conditionals.
Write a loop that prints all even numbers up to 100.
Use switch to create a text-based menu.

⚙️ Pro Tip

Think like the compiler: what executes first, what repeats, and when does it stop?
Drawing quick flow diagrams can help lock this in before you start writing.

🧩 Phase 3: Methods and Modular Design (Modules 5–6)
🎯 Learning Targets

Write reusable methods (static and non-static)
Understand parameters and return values
Pass data between methods
Method overloading

💻 Practice Tasks

Write a sumArray() method that adds all elements in an array.
Refactor your calculator into multiple methods.
Write and test a recursive method (like factorial or Fibonacci).

⚙️ Pro Tip

Start using IntelliJ’s debugger to step through your code — this will help when you hit harder logic later in DSA courses.

🧩 Phase 4: Object-Oriented Programming (Modules 7–9)
🎯 Learning Targets

Create and use classes and objects
Understand constructors
Use getters/setters (this keyword)
Practice encapsulation and inheritance

💻 Practice Tasks

Create a Pet class with fields like name, species, and age.
Create a Fish subclass that extends Pet and adds waterType or tankSize.
Instantiate objects and call their methods.
Override toString() and equals() methods.

⚙️ Pro Tip

Relate your OOP design back to your D427 mindset — you already think in entities and relationships. Classes are just “table blueprints with behavior.”

🧩 Phase 5: Arrays and File I/O (Modules 10–12)
🎯 Learning Targets

Work with arrays and ArrayList
Read from and write to files
Handle exceptions with try / catch

💻 Practice Tasks

Write a program that stores names in an array and prints them.
Read a .txt file line by line and count the words.
Create a contact list program that saves entries to a file.

⚙️ Pro Tip

Make sure you can explain how Java stores arrays (fixed-size) vs. ArrayLists (dynamic) — this concept comes back hard in later courses.

🧩 Phase 6: Final Project (Performance Assessment)
🎯 Common Structure

You’ll write a small, menu-driven application — examples from past students include:

Inventory Manager (CRUD items)
Bank Account Manager
Student Grade Tracker
Pet Adoption System

It usually requires:

2+ classes
Constructors
Loops, conditionals, and arrays
File input/output or basic persistence

🧠 Strategy

Build iteratively — start with core classes and print output before adding logic.
Use IntelliJ’s console for testing methods step by step.
Comment liberally to show clarity (graders love readable code).
